## 📌 프로젝트 소개

이 프로젝트는 **이젠기업 웹사이트 리뉴얼 퍼블리싱 작업**입니다.
기존 사이트를 참고하되 **디자인은 100% 새롭게 기획 및 제작**했으며,
HTML, CSS, JavaScript를 기반으로 한 **정적 웹사이트**로 구현했습니다.
메인 비주얼 영역에는 Swiper 플러그인을 적용해 시각적 완성도와 사용성을 개선했습니다.

---

## 🛠 기술 스택

-   **HTML5**

    -   시맨틱 마크업을 활용한 구조 설계

-   **CSS3**

    -   공통 UI 스타일링
    -   페이지별 스타일 관리

-   **JavaScript (Vanilla JS)**

    -   DOM 제어
    -   사용자 이벤트 처리

-   **Swiper**

    -   메인 비주얼 및 콘텐츠 슬라이드 구현

---

## ✨ 핵심 기능

-   메인 비주얼 영역 Swiper 슬라이드 적용

    -   자동 재생 및 반복 슬라이드
    -   드래그/터치 기반 슬라이드 전환

-   공통 레이아웃 구성

    -   Header / Footer UI 구성
    -   메뉴 이동을 위한 기본 내비게이션

-   JavaScript 기반 인터랙션

    -   클릭 이벤트 처리
    -   클래스 제어를 통한 UI 상태 변경
